# 多人位置共享功能使用指南

## 🎯 功能介绍

多人位置共享功能允许多个用户同时在地图上看到彼此的实时位置，非常适合：
- 🏃 团队跑步训练
- 👥 朋友约跑
- 🎉 马拉松活动组织
- 📍 实时位置追踪

## 🚀 快速开始

### 第一步：配置 Firebase

1. **创建 Firebase 项目**
   - 访问 [Firebase Console](https://console.firebase.google.com/)
   - 点击"添加项目"
   - 输入项目名称（如：`beijing-marathon-tracker`）
   - 创建项目

2. **创建 Web 应用**
   - 在项目中点击 Web 图标 `</>`
   - 输入应用昵称
   - 复制 Firebase 配置代码

3. **启用 Realtime Database**
   - 左侧菜单 → "Realtime Database"
   - 点击"创建数据库"
   - 选择位置：`asia-southeast1`（新加坡）
   - 选择"测试模式"启动

4. **设置安全规则**
   ```json
   {
     "rules": {
       "users": {
         "$userId": {
           ".read": true,
           ".write": true
         }
       }
     }
   }
   ```

5. **更新配置**
   - 打开 `index.html` 文件
   - 找到 Firebase 配置部分（约第 10 行）
   - 替换为你的配置信息：
   ```javascript
   const firebaseConfig = {
     apiKey: "你的API密钥",
     authDomain: "你的项目.firebaseapp.com",
     databaseURL: "https://你的项目-default-rtdb.firebaseio.com",
     projectId: "你的项目ID",
     storageBucket: "你的项目.appspot.com",
     messagingSenderId: "你的发送者ID",
     appId: "你的应用ID"
   };
   ```

### 第二步：使用多人共享

1. **打开网站**
   - 访问 https://pfliu-nlp.github.io/app/

2. **输入昵称**
   - 在"输入你的昵称"框中输入你的名字
   - 例如：小明、Runner01 等

3. **开始共享**
   - 点击"🌐 开始共享位置"按钮
   - 允许浏览器访问位置信息
   - 系统会自动开始定位

4. **查看其他用户**
   - 在线用户列表会显示所有正在共享位置的用户
   - 地图上会显示不同颜色的标记代表不同用户
   - 点击标记可以查看用户详细信息

5. **停止共享**
   - 点击"🔴 停止共享"按钮
   - 你的位置将从地图上移除

## 🔒 隐私和安全

### 数据隐私
- ✅ 位置数据仅在共享期间可见
- ✅ 停止共享后数据立即删除
- ✅ 不保存历史位置记录
- ✅ 只显示昵称，不显示真实姓名

### 安全建议
- 💡 使用昵称而非真实姓名
- 💡 只在需要时开启共享
- 💡 跑步结束后记得停止共享
- 💡 不要在不安全的网络环境下使用

## ⚡ 性能优化

### 电池消耗
- 📱 使用高精度 GPS 会增加电池消耗
- 💡 建议充满电后使用
- 💡 可以携带充电宝

### 网络流量
- 📊 每次位置更新约 200 字节
- 📊 每分钟约 12KB 流量
- 📊 1 小时约 720KB（不到 1MB）

### 数据库限制
- 📦 Firebase 免费套餐：
  - 1GB 存储空间
  - 10GB/月 下载量
  - 100 并发连接
- 💡 对于这个应用完全够用

## 🐛 常见问题

### Q: 为什么看不到其他用户？
A: 可能原因：
1. 其他用户还没有开启共享
2. Firebase 配置不正确
3. 网络连接问题
4. 检查浏览器控制台是否有错误

### Q: 位置更新很慢怎么办？
A: 
1. 检查网络连接
2. 确保 GPS 信号良好（户外效果最佳）
3. 刷新页面重新连接

### Q: 如何删除我的位置数据？
A: 
1. 点击"停止共享"按钮
2. 数据会立即从数据库删除
3. 关闭浏览器标签页也会自动删除

### Q: 可以同时有多少人在线？
A: 
- Firebase 免费套餐支持 100 并发连接
- 对于一般使用完全足够
- 如需更多，可以升级到付费套餐

### Q: 位置精度如何？
A: 
- 户外 GPS：5-20 米
- 室内/城市峡谷：20-100 米
- 取决于设备和环境
