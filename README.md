# 🏃‍♂️ 北京二环马拉松路线规划器

一个基于百度地图的北京二环马拉松路线规划和实时定位工具。

**🌐 在线访问：** https://pfliu-nlp.github.io/app/

---

## ✨ 主要功能

### 1. 📍 实时GPS定位追踪 ⭐ 新功能
- 🔄 **持续追踪**：实时显示你在路线上的位置
- 📏 **距离计算**：自动计算到最近节点的距离
- ✅ **状态提示**：在路线上/接近路线/偏离路线
- 🎯 **高精度模式**：使用GPS高精度定位
- 📊 **精度显示**：显示GPS定位精度范围
- 🔵 **可视化标记**：蓝色圆点+精度圈

### 2. 🔍 地点搜索功能
- 🗺️ **智能搜索**：搜索北京任意地点
- 📋 **多结果显示**：最多显示5个搜索结果
- 🔵 **自定义标记**：蓝色标记区别于路线
- 🎬 **动画效果**：弹跳动画吸引注意
- ⌨️ **快捷操作**：支持回车键搜索

### 3. 🗺️ 完整路线展示
- 📏 **总距离**：33公里（实际跑友验证）
- 🌳 **绿道覆盖**：80%路段有专业绿道
- 🚪 **15个关键节点**：西直门→德胜门→...→阜成门
- ⚠️ **5个易错点**：特别标注需要注意的位置
- 🎨 **颜色区分**：不同颜色标记不同类型点位

### 4. 🚻 公共设施标注
- 📍 **13个公厕**：平均每2-3公里一个
- 🏛️ **文化地标**：明城墙遗址、古观象台等
- 🌳 **公园绿地**：龙潭公园、陶然亭公园等
- 🚇 **地铁站点**：方便跑前跑后交通

### 5. 📝 详细路线信息
- 🛣️ **9段详细描述**：每段特点、距离、路面情况
- 💡 **实用建议**：基于真实跑友经验
- ⏰ **最佳时间**：早上5:30-7:00
- 🏃 **配速策略**：分段配速建议
- 🎒 **装备建议**：补给、装备、安全提示

## 🛣️ 路线详情

### 总体信息
- **总距离**：33公里
- **起点/终点**：西直门
- **方向**：顺时针（与车流同向）
- **绿道覆盖**：约80%
- **关键节点**：15个城门

### 路线分段（顺时针）
1. **西直门 → 德胜门** (2.5km) - 西二环绿道起点
2. **德胜门 → 安定门** (2.5km) - 北护城河美景
3. **安定门 → 雍和宫** (1.5km) - 文化氛围浓厚
4. **雍和宫 → 东直门** (2km) - 继续沿护城河
5. **东直门 → 建国门** (5km) - 东二环长段
6. **建国门 → 广渠门** (3km) - 明城墙遗址
7. **广渠门 → 左安门** (4km) - 南二环东段
8. **左安门 → 右安门** (5km) - 南二环中段
9. **右安门 → 广安门** (2km) - 南二环西段
10. **广安门 → 复兴门** (2km) - 接近终点
11. **复兴门 → 阜成门** (1.5km) - 西二环南段
12. **阜成门 → 西直门** (2km) - 回到起点

### 关键节点（15个）
西直门 → 德胜门 → 安定门 → 雍和宫桥 → 东直门 → 东便门 → 建国门 → 广渠门 → 左安门 → 木樨园 → 右安门 → 广安门 → 复兴门 → 阜成门 → 西直门

### 易错点（5个）
1. **小街桥下** (1km) - 桥下通道昏暗
2. **雍和宫桥** (4km) - 注意下到河堤绿道
3. **东便门桥后** (15km) - 关键绿道入口
4. **广渠门桥后** (17km) - 下行到绿道
5. **右安门到广安门** (23-25km) - 连续绿道段

## 💡 实用建议

### 最佳跑步时间
- ⏰ **早上5:30-7:00** - 避开车流高峰，空气质量好
- 📅 **工作日优于周末** - 人流较少

### 路面选择
- 🌳 **80%绿道** - 塑胶跑道，保护膝盖
- 🛣️ **20%人行道** - 硬质步道

### 补给准备
- 💧 **水壶**：500ml
- ⚡ **能量胶**：2-3支
- 🧂 **盐丸**：适量
- 📍 **补给点**：12km（建国门）、24km（广安门）

### 配速策略
- **前10km**：5:20-5:30/km（红绿灯多）
- **中段10-25km**：5:00-5:15/km（绿道为主）
- **最后8km**：根据体力调整

### 安全注意
- 💡 **桥下通道**：光线昏暗，建议佩戴头灯
- 🚶 **绿道规则**：左快右慢，礼让行人
- 📱 **保持通讯**：手机充满电或带移动电源

### 装备建议
- 👟 **跑鞋**：缓冲性好的跑鞋
- 👕 **服装**：透气速干
- 🌞 **夏季**：防晒+补水
- ❄️ **冬季**：保暖+防风

## 🔧 技术实现

### 前端技术
- **HTML5**：结构和语义化
- **CSS3**：渐变、动画、响应式设计
- **JavaScript**：交互逻辑

### 地图API
- **百度地图 JavaScript API**
  - Map API - 地图显示
  - Marker API - 标记点
  - Polyline API - 路线绘制
  - LocalSearch API - 地点搜索
  - Circle API - 精度圈

### 定位技术
- **HTML5 Geolocation API**
  - `watchPosition` - 持续追踪
  - `enableHighAccuracy` - 高精度模式
  - 距离计算算法
  - 实时状态判断

### 部署
- **GitHub Pages**
- **HTTPS** - 安全访问
- **响应式设计** - 适配手机和桌面
