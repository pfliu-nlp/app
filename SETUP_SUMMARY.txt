========================================
  接下来你需要做的：
========================================

步骤 1: 创建 Firebase 项目
--------------------------
1. 访问 https://console.firebase.google.com/
2. 点击"添加项目"
3. 输入项目名称（如：beijing-marathon-tracker）
4. 创建项目

步骤 2: 创建 Web 应用
--------------------------
1. 在项目中点击 Web 图标 </>
2. 输入应用昵称
3. 复制 Firebase 配置代码

步骤 3: 启用 Realtime Database
--------------------------
1. 左侧菜单 → "Realtime Database"
2. 点击"创建数据库"
3. 选择位置：asia-southeast1（新加坡）
4. 选择"测试模式"启动
5. 点击"启用"

步骤 4: 设置安全规则
--------------------------
在 Realtime Database 的"规则"标签中，粘贴：

{
  "rules": {
    "users": {
      "$userId": {
        ".read": true,
        ".write": true
      }
    }
  }
}

点击"发布"保存。

步骤 5: 更新配置文件
--------------------------
方法 A - 使用脚本（推荐）：
  ./update_firebase_config.sh

方法 B - 手动编辑：
  编辑 index.html 和 beijing_second_ring_marathon.html
  找到 firebaseConfig 部分（约第 820 行）
  替换为你的 Firebase 配置

步骤 6: 提交并部署
--------------------------
git add .
git commit -m "Configure Firebase for multiplayer"
git push origin main

等待 1-2 分钟后访问：
https://pfliu-nlp.github.io/app/

========================================
  使用方法
========================================

单人模式：
1. 打开网站
2. 点击"📍 定位"
3. 允许位置权限
4. 开始跑步

多人共享模式：
1. 打开网站
2. 输入昵称
3. 点击"🌐 开始共享位置"
4. 允许位置权限
5. 邀请朋友也打开网站
6. 在地图上看到彼此的位置

========================================
  常见问题
========================================

Q: 看不到其他用户？
A: 检查 Firebase 配置是否正确，
   确保其他用户也开启了共享

Q: 如何停止共享？
A: 点击"🔴 停止共享"按钮

Q: 数据会被保存吗？
A: 不会，停止共享后数据立即删除

Q: 免费吗？
A: Firebase 免费套餐完全够用
   支持 100 并发用户
